japanese-otf-uptex

japanese-otf package "otfbeta (v1.7b8 2019/04/01)" by psitau-san
( http://psitau.kitunebi.com/otf.html )


The source files of japanese-otf-uptex are maintained
at this repository.
See the file "japanese-otf-uptex/00otf-uptex.txt"
for more details.
